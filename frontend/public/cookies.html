Política de cookies y almacenamiento local

Resumen: esta web utiliza únicamente mecanismos técnicos mínimos y orientados al funcionamiento básico del servicio.

1. Qué utiliza esta web

Apagones Ciudadanos utiliza almacenamiento local del navegador y mecanismos técnicos equivalentes estrictamente necesarios para el funcionamiento básico del servicio.

2. Finalidad

Estos mecanismos técnicos sirven para mantener coherencia operativa, reducir duplicados y facilitar el funcionamiento básico del servicio.

3. Qué no hace esta web

En esta versión no se pretende utilizar cookies o tecnologías equivalentes con fines de publicidad comportamental ni perfiles comerciales.

4. Componentes de terceros

Determinados recursos técnicos de terceros, como servicios cartográficos, pueden implicar solicitudes técnicas a sus respectivos servidores.
