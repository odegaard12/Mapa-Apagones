Política de privacidad

Mapa Apagones está diseñado con minimización de datos: sin cuentas, sin CUPS, sin texto libre, sin fotos y sin publicación de viviendas exactas.

Responsable y contacto

Para consultas relacionadas con privacidad puedes escribir a privacidad@mapa-apagones.es.

Qué datos trata la plataforma

La plataforma está diseñada para tratar únicamente información técnica y operativa necesaria para mostrar incidencias agregadas y prevenir abusos. Según la funcionalidad activa, pueden tratarse:

  • Tipo de incidencia reportada: sin luz, microcortes, baja tensión o ya volvió.
  • Zona aproximada, celda o ámbito geográfico agregado.
  • Fecha y hora técnica del reporte o actualización.
  • Estado agregado de la zona y número de confirmaciones.
  • Identificador técnico local del navegador, almacenado localmente, para reducir duplicados.
  • Hashes técnicos derivados del identificador local y/o de la IP para rate limiting, anti-abuso y coherencia operativa.
  • Registros técnicos mínimos necesarios para seguridad, diagnóstico y mantenimiento.

Qué datos no se solicitan

  • No se crean cuentas de usuario.
  • No se solicita CUPS.
  • No se solicita nombre, DNI, teléfono ni datos de contrato eléctrico.
  • No se solicita dirección exacta.
  • No se permiten fotos ni texto libre en los reportes.
  • No se publica la ubicación exacta de viviendas.

Finalidad

Los datos se usan para mostrar avisos ciudadanos agregados por zona, evitar duplicados, limitar abuso o spam, mantener la seguridad técnica del servicio y mejorar la fiabilidad operativa.

Turnstile y protección anti-abuso

En producción, el formulario de reporte puede usar Cloudflare Turnstile para verificar que el envío no procede de automatizaciones abusivas. El token de Turnstile se usa para validación y no debe almacenarse en la base de datos de reportes.

Conservación

Los reportes están pensados para ser temporales y caducar pasado un periodo operativo. Los registros técnicos de seguridad o anti-abuso se conservarán solo durante el tiempo razonablemente necesario para proteger el servicio.

Terceros técnicos

La web puede usar Cloudflare para alojamiento, seguridad, DNS, Pages, Turnstile o enrutamiento, y OpenStreetMap/Leaflet para la visualización cartográfica. Estos servicios pueden recibir solicitudes técnicas necesarias para entregar la web o los mapas.

Derechos y consultas

Para cualquier consulta razonable sobre privacidad, escribe a privacidad@mapa-apagones.es. Al no existir cuentas ni perfiles personales, algunas solicitudes pueden depender de la información técnica disponible para identificar un reporte concreto.
