# Apagones Ciudadanos

Mapa ciudadano de apagones basado solo en reportes de usuarios.

## Principios de esta v1
- Sin cuentas
- Sin CUPS
- Sin datos de eléctricas
- Sin texto libre
- Sin fotos
- OpenStreetMap + reportes ciudadanos

## Arranque
docker compose up --build -d
